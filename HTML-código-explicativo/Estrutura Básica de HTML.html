O que é HTML?

A abreviatura HTML significa Hypertext Markup Language, que em português podemos traduzir literalmente como Hypertext Markup Language, refere-se a um método de configuração de páginas da Internet.

Estrutura Básica de HTML

Como é?

A estrutura básica de html é composta por nove tags, onde cada uma possui uma funcionalidade. Abaixo temos uma imagem representando a estrutura básica de html:

< !DOCTYPE html >

A primeira tag é < !DOCTYPE html > e é usada para informar o navegador que o código está em html.

< html lang="pt-br" >

Essa tag tem a função de disitinguir o idioma da página, caso a tag fosse < html lang="en" >, o idioma da página seria em inglês.

< head >

A tag é usada como cabeçalho e é responsável por fazer o navegador interpretar informações iniciais do código, como por exemplo, o título da página dentro dessa tag e os metadados que serão carregados na página. É importante lembrar que o conteúdo da tag não fica visível no site.

< tittle >

Essa tag é usada para definir o título da página, na qual fica será exibido no resultado de pesquisa de uma guia, como na imagem abaixo mostra.

< meta charset=!utf-8" >

É uma tag que define o conjunto de caracteres usado pelo documento. O conjunto de caracteres "UTF-8" é um padrão de codificação que permite a representação de uma variedade de caracteres, incluindo caracteres latinos, caracteres não latinos (como caracteres chineses, japoneses e coreanos) e símbolos especiais.

< body >

Na tag < body > todo o conteúdo será visualizado no site estará dentro dessa tag, representando justamente o corpo da página.
